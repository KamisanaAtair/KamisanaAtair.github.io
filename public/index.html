🔧 API 接口文档

GET /api/stats - 获取访问统计数据
返回：总访问数、独立IP数、今日访问数等
GET /api/logs - 获取访问日志
参数：limit=数量, offset=偏移量
POST /api/visit - 记录一次访问
自动记录IP、User-Agent、时间戳等信息
GET /api/ip-stats - 获取IP访问统计
返回各IP的访问次数排行
DELETE /api/logs - 清空访问日志
需要管理员权限
